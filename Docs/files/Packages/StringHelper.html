compareString (target, compareTo, ignoreCase)
Indicates whether two strings are identical, optionally ignoring case and safely handling nil values. If either object is not nil and is not a string, returns false.

Parameters

  • target: The first string to compare.
  • compareTo: The second string to compare.
  • ignoreCase: Determines whether case is ignored during comparison.

Usage

  • compareString( nil, nil ) -- true
  • compareString( "Hi", "hi" ) -- false
  • compareString( "Hi", "hi", true ) -- true
  • s = {}; compareString( s, s ) -- false

Return value:

True if both strings are nil or are equal; false otherwise.
endsWith (target, suffix, ignoreCase)
Indicates whether the specified target string ends with the specified suffix string, optionally ignoring case and safely handling nil values. If either object is not nil and is not a string, returns false.

Parameters

  • target: The target string to be examined for whether it ends with the specified suffix string.
  • suffix: The string to be looked for at the end of the target string.
  • ignoreCase: Determines whether case is ignored during evaluation.

Usage

  • endsWith( nil, nil ) -- true
  • endsWith( "Hi there", "Here" ) -- false
  • endsWith( "Hi there", "Here", true ) -- true
  • endsWith( "Hi there", "" ) -- true
  • s = "a"; endsWith( s, s ) -- true
  • s = {}; endsWith( s, s ) -- false

Return value:

True if both strings are nil, or if the specified target string ends with the specified suffix string, or if the specified suffix string is an empty string; false otherwise.
