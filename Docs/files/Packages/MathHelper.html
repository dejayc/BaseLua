roundNumber (number, decimalPlaces)
Returns the simply rounded representation of the specified number.

Parameters

  • number: The number to round.
  • decimalPlaces: The number of decimal places to which to round the specified number. Defaults to 0.

Usage

  • roundNumber( 98.4 ) -- 98
  • roundNumber( 98.6 ) -- 99
  • roundNumber( 98.625, 1 ) -- 98.6
  • roundNumber( 98.625, 2 ) -- 98.63

Return value:

The simply rounded representation of the specified number.
