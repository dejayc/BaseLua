Functions

printf (format, ...)
Prints the provided parameters via 'print', formatted according to the specified format, following the formatting convention of 'string.format'. Automatically prints an end-of-line character following the string.

Parameters

  • format: The 'string.format' formatting convention to apply to the provided parameters.
  • ...: Optional parameters to format according to the specified format.

Usage:

printf( "Hello, %s!", "world" )
writef (format, ...)
Writes the provided parameters via 'io.write', formatted according to the specified format, following the formatting convention of 'string.format'. Does not automatically print an end-of-line character following the string.

Parameters

  • format: The 'string.format' formatting convention to apply to the provided parameters.
  • ...: Optional parameters to format according to the specified format.

Usage:

writef( "Hello, %s!", "world" )

Fields

  • PACKAGE_PATH_SEPARATOR: Indicates the character that separates Lua nested package paths that appear in invocations to the Lua "require" function. Statically assigned the value "."
  • PACKAGE_NAME_PATTERN: The Lua pattern that can be used to identify the package name in nested package paths that appear in invocations to the Lua "require" function.
  • PACKAGE_PATH_PATTERN: The Lua pattern that can be used to identify the leading paths in nested package paths that appear in invocations to the Lua "require" function.

BaseLua package helper that facilitates working with BaseLua packages.

Fields

  • name: The package name for this BaseLua class. Usually should be 'BaseLua', unless the file name of this class has been changed.
  • path: The package path for all BaseLua classes.
