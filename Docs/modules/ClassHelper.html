autoclass (name, object, ...)
Invokes the 'extend' method of the 'BaseClass' class, effectively allowing a BaseLua subclass to be created with the specified name.

Parameters

  • name: The name to assign to the new class.
  • object: An optional object to be cast to the new class.
  • ...: Optional parameters to be passed to the 'extend' method of the BaseLua 'BaseClass' class.

Usage

  • local myClassInstance = ClassHelper.autoclass( "myClassName" )
  • local myClassInstance = ClassHelper.autoclass( "myClassName", existingClass )

Return value:

The valued returned by the 'extend' method of the BaseLua 'BaseClass' class, usually expected to be a new class that subclasses 'BaseClass'.

See also:

  • autoclass
  • BaseClass:extend
autoextend (target, name, object, ...)
Invokes the 'extend' method of the specified class, to create a subclass of the specified class with the specified name.

Parameters

  • target: The class upon which to invoke the 'extend' method. If a string, will be interpreted as a file name to be 'required' first, which will then be expected to return a class with an 'extend' method to invoke.
  • name: The name to assign to the new subclass.
  • object: An optional object to be cast to an instance of the subclass of the specified class.
  • ...: Optional parameters to be passed to the 'extend' method.

Usage

  • local mySubClassInstance = ClassHelper.autoextend( myClassInstance, "MySubClassName" )
  • local mySubClassInstance = ClassHelper.autoextend( "myClassName", "MySubClassName" )

Return value:

The valued returned by the 'extend' method of the specified class, usually expected to be a new class that subclasses the specified class.

See also:

  • BaseClass:extend

class (...)
Invokes the 'extend' method of the BaseLua 'BaseClass' class, effectively allowing a BaseLua subclass to be created. Use this method if you are passing in, as the first parameter, a table to be converted into a class. The table should contain a property 'class.name' that defines the name of the resulting class. If you want to create a new class from scratch, and would rather pass in the class name as a parameter, please see function 'autoclass'.

Parameters

  • ...: Optional parameters to be passed to the 'extend' method of the BaseLua 'BaseClass' class.

Usage:

local myClassInstance = ClassHelper.class( "myClassName" )

Return value:

The valued returned by the 'extend' method of the BaseLua 'BaseClass' class, usually expected to be a new class that subclasses 'BaseClass'.

See also:

  • autoclass
  • BaseClass:extend
