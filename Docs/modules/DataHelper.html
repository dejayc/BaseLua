getNonNil (...)
Returns the first non-nil value from the specified list of values.

Parameters

  • ...: A list of values to search for the first non-nil value.

Usage

  • getNonNil( nil, 5, nil ) -- 5
  • getNonNil( "Hi", nil, 3 ) -- "Hi"

Return value:

The first non-nil value.
hasValue (value)
Indicates whether the specified target value is not nil and is not an empty string.

Parameters

  • value: The target value to evaluate.

Usage

  • hasValue( nil ) -- false
  • hasValue( 5 ) -- true
  • hasValue( "" ) -- false
  • hasValue( " " ) -- true

Return value:

True if the specified target value is not nil and is not an empty string; otherwise, false.
ifThenElse (_if, _then, _else)
Returns the specified target value if the specified boolean test parameter is true; otherwise, returns the specified default value.

Parameters

  • _if: The boolean test parameter that determines whether the target value or default value is to be returned; true if the target value is to be returned, false if the default value is to be returned.
  • _then: The target value to be returned if the specified boolean test parameter is true.
  • _else: The default value to be returned if the specified boolean test parameter is false.

Usage

  • ifThenElse( true, 1, 0 ) -- 1
  • x = 3; y = 9; ifThenElse( x > y, x, y ) -- 9

Return value:

The specified target value if the specified boolean test parameter is true; otherwise, the specified default value.
